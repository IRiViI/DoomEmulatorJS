Description

The engine of this game is fully run on a neural network. The inputs (left, right and shoot) goes together with the a latent space vectors of previous frames into an recurrent neural network to predict the next latent space vector. The newly generated vector is then decoded by a convolution network to construct a 640x480 frame.
