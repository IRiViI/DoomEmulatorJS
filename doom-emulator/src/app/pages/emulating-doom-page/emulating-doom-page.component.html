Description

The engine of this game is fully running on a neural network trained with deep learning. The inputs (left, right and shoot) goes together with the latent space vectors of previous frames into a recurrent neural network in order to predict the next latent space vector. The newly generated vector is then decoded by a convolution network to construct a 640x480 frame.
